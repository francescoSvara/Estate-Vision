# EV Dashboard - PMTiles Viewer

Interactive geospatial web application for visualizing Italian cadastral parcels, census data, and real estate market information.

## Overview

A modern MapLibre GL JS viewer built with Vite and vanilla JavaScript, featuring modular UI components, multi-language support, and efficient PMTiles vector tile delivery for professional real estate data visualization.
